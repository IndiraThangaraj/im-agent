# Issue Management Agent

1.  Put your `.pdf` files in the `knowledge_base/` folder.
2.  Run the docker compose file.
4.  Import `n8n-workflows/Im_agent.json` into n8n.
5.  Set up n8n Credentials for **Postgres**, **Embedding Model**, and **Chat Model**.
6.  Link those credentials in the workflow and activate it.
